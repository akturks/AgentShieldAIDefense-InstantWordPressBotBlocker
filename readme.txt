Protect your WordPress site from aggressive AI crawlers and indirect prompt injection attacks.

== Description ==

AgentShield AI Defense is a specialized security plugin designed to protect your digital assets from AI-driven threats. It implements several layers of defense:

1. **Content Cleaner**: Removes invisible Unicode characters used in indirect prompt injection.
2. **AI Honeypot**: Lures bots into a "tarpit" to slow them down and consume their resources.
3. **Bot Blocker**: Immediately blocks known AI crawlers via User-Agent detection.
4. **Invisible Watermarking**: Marks your content to identify data scraped by AI models.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure settings in the 'AgentShield AI' menu.
